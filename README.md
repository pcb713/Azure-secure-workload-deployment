# Secure Azure Workload Deployment

## Scenario

A company requires a secure Azure environment for an internal
application. The infrastructure must follow cloud security
best practices including network segmentation and role-based access.

## Solution

This project deploys a secure Azure workload using:

- Infrastructure as Code (Bicep)
- Azure Virtual Network
- Resource Group isolation
- Secure deployment automation

## Technologies Used

- Microsoft Azure
- Azure CLI
- Bicep
- PowerShell

## Architecture

The deployment creates a secure Azure environment with a
virtual network and isolated resource group.
